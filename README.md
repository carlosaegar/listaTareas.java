Ejercicio 1, archivo BC3 II, REALIZADO EN CLASE Enunciado; 1. Listas: Lista de tareas con prioridad
Crea un programa que gestione una lista de tareas con ArrayList<String>.
El programa debe permitir, mediante un menú sencillo:
• 1: Añadir una tarea al final.
• 2: Insertar una tarea en una posición concreta (prioridad).
• 3: Eliminar una tarea por posición.
• 4: Mostrar todas las tareas numeradas.
• 0: Salir.
Si la posición indicada no es válida (fuera de rango), mostrar un mensaje de error y no hacer nada.
